# 项目情况介绍文档

## 1. 项目背景
本项目是一个基于 Web 技术的 **浙江省 3D 可视化地图** 系统。旨在通过高保真的 3D 渲染技术，将地理信息数据以"暗黑科技风"的形式进行展示，为决策者、管理者提供直观的数据大屏视觉组件。

## 2. 核心价值
*   **高沉浸感**：利用 Babylon.js 引擎实现流畅的 3D 场景漫游。
*   **科技美学**：采用深青色系（Dark Teal）为主调，结合自发光（Emissive）和辉光（Glow）特效，打造未来感视觉体验。
*   **强交互性**：支持鼠标点击交互，实现地块抬升、高亮聚焦、相机自动运镜等动态反馈。
*   **数据驱动**：基于 GeoJSON 标准地理数据生成 3D 模型，易于扩展和维护。

## 3. 适用场景
*   智慧城市指挥中心大屏
*   数据可视化仪表盘背景组件
*   政务汇报演示系统

## 4. 现状概述
项目目前已完成核心功能的开发，并已将地图数据源切换为**浙江省**，实现了地市级 3D 建模、材质渲染、交互逻辑以及多种视觉特效（光柱、旋转光圈、上升粒子等）。代码结构清晰，基于 Vue 3 框架，便于后续集成到现有业务系统中。

## 5. 关键配置

| 参数 | 当前值 | 说明 |
|------|--------|------|
| 数据源 | `浙江省.geojson` | 市级行政区划 |
| 底图自发光色 | `#2C8080` | 暗处亮度 |
| 高亮色 | `#22FFFF` | 选中状态颜色 |
| 光柱缩放 | 0.45 | 初始大小（原尺寸 30%） |
| 摄像机位置 | (120.98, 4.23, 23.13) | 初始视角 |
| 摄像机目标 | (120.64, -0.58, 28.96) | 观察焦点 |
