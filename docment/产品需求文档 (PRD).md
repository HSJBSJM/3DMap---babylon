# 产品需求文档 (PRD)

**项目名称**: 浙江省 3D 可视化地图组件
**版本**: v1.1
**日期**: 2026-05-08

## 1. 概述
### 1.1 产品目标
构建一个具备高质量视觉效果和流畅交互体验的 3D 地图组件，用于展示浙江省地理空间数据。组件需具备科技感，支持用户通过点击与地图元素互动，并提供动态的视觉反馈。

### 1.2 用户画像
*   **大屏浏览者**: 需快速获取浙江省各地市的地理位置分布和关键信息。
*   **系统操作员**: 通过点击操作查看特定地市的详细状态。

## 2. 功能需求

### 2.1 地图渲染
*   **F01-基础模型**: 解析浙江省 GeoJSON 数据，生成具有厚度的 3D 地市地块模型。
*   **F02-纹理映射**: 统一使用整张地图纹理进行全局 UV 映射，确保视觉连贯性。
*   **F03-视觉风格**:
    *   **主色调**: 深青色 (`#0E4040`)。
    *   **自发光**: 适度自发光 (`#2C8080`)，确保在暗色背景下清晰可见。
    *   **边界**: 地块边缘需有清晰的白色边界线（上层）和暗色边界线（下层）。

### 2.2 交互逻辑
*   **F04-点击选中**:
    *   **高亮反馈**: 选中地块变为高亮青色 (`#22FFFF`)，并带有辉光效果。
    *   **动态抬升**: 选中地块沿 Y 轴平滑抬升 `0.2` 单位。
    *   **背景暗化**: 未选中地块亮度降低，材质变暗 (`#051515`)，以突出主体。
*   **F05-相机运镜**:
    *   点击地市后，摄像机平滑移动（Pan & Zoom）至目标地市上方，以最佳视角展示。
*   **F06-状态复原**:
    *   再次点击已选中地块，所有状态复原（模型回落、亮度恢复、光柱恢复显示）。

### 2.3 场景元素
*   **F07-光柱标记**:
    *   每个地市中心显示垂直光柱 + 标注点 + 标注光圈。
    *   **自适应缩放**: 根据摄像机距离自动调整光柱粗细（最近不小于 0.2，最远不小于 0.3）。
    *   **智能显隐**: 选中某地市时，自动隐藏其他地市的光柱，避免视觉干扰。
    *   **初始大小**: 光柱缩放系数为 `0.45`（原尺寸 30%），适配初始视角。
*   **F08-文字标签**:
    *   地市名称标签悬浮于地表 (`y=0`)，文字清晰无遮挡。
*   **F09-环境装饰**:
    *   底部设有旋转的光圈和点阵层，增加场景的动态感和层次感。
    *   装饰层需保持 100% 不透明度，确保视觉强度。
*   **F10-上升粒子**:
    *   地图周围环绕着随机生成的上升粒子，模拟能量溢出的科技氛围。
    *   **动态效果**: 粒子具有 3 秒生命周期，包含淡入、持续、淡出三个阶段。
    *   **视觉细节**: 粒子带有辉光效果，且分布范围严格限制在装饰点阵层以内。

### 2.4 辅助功能
*   **F11-摄像机调试信息**:
    *   **实时数据显示**: 屏幕右上角显示摄像机的位置 (Position)、目标点 (Target) 和旋转角度 (Rotation)。
    *   **显隐控制**: 屏幕底部提供开关按钮，用于切换调试信息的显示状态。

## 3. 非功能需求
*   **性能**: 在主流 PC 浏览器上保持 60 FPS 流畅运行。
*   **兼容性**: 适配 Chrome、Edge 等现代浏览器。
*   **可配置性**: 颜色、材质、摄像机参数需在代码中集中管理，便于后期调整。
